# MindFlow Mind Map App

This is a client-side React application with an Express backend for serving assets.

## Running with Docker (Recommended)

If you have Docker installed, you can run the entire app with a single command:

```bash
docker-compose up --build
```

The app will be available at `http://localhost:3000`.

## Running Locally (Node.js)

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open the app:**
   Go to `http://localhost:3000` in your browser.

## Portable Version

If you don't want to install anything, you can just double-click the `portable_mindflow.html` file to open it in any browser. Note that this is a "lite" version of the app.
