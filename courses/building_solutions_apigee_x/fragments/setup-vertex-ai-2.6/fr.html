1. Dans le menu de navigation de Google Cloud Console, cliquez sur Vertex AI > Workbench.

  2. Sur la page "Instances de notebook", cliquez sur Nouveau notebook > TensorFlow Enterprise > TensorFlow Enterprise 2.6 (avec LTS) > Sans GPU.

  3. Dans la boîte de dialogue Nouveau notebook, vérifiez le nom de la VM deep learning. Si vous ne souhaitez pas modifier la région ni la zone, laissez tous les paramètres tels quels, puis cliquez sur Créer. Le démarrage de la nouvelle VM prend deux à trois minutes.

  4. Cliquez sur Ouvrir JupyterLab.
    Une fenêtre JupyterLab s'ouvre dans un nouvel onglet.

  5. Lorsque le pop-up "Compilation recommandée" s'affiche, cliquez sur Compiler. Si le message "Échec de la compilation" apparaît, ignorez-le.
