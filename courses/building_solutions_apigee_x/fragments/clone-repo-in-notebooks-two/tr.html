JupyterLab örneğinizdeki mlops-on-gcp not defterini klonlamak için:

1. Adım

JupyterLab'de, Terminal simgesini tıklayarak yeni bir terminal açın.

2. Adım

Komut satırı isteminde aşağıdaki komutu yazıp Enter'a basın.

git clone https://github.com/GoogleCloudPlatform/mlops-on-gcp Not

Klonlanan depo JupyterLab kullanıcı arayüzünde görünmüyorsa üstteki menüden Git --> Depo klonla'yı tıklayarak kullanıcı arayüzünde depoyu (https://github.com/GoogleCloudPlatform/mlops-on-gcp) klonlayabilirsiniz.

3. Adım

Depoyu klonlayabildiğinizden emin olmak için mlops-on-gcp dizinini çift tıklayın ve dizin içeriğini görüp göremediğinizi kontrol edin. Bu kurstaki Jupyter not defteri tabanlı laboratuvarlarda kullanılan dosyaların tamamı bu dizindedir.
