Siga estos pasos para clonar el notebook mlops-on-gcp en su instancia de JupyterLab:

  1. En JupyterLab, haga clic en el ícono de terminal para abrir una terminal nueva.

  2. En la ventana de línea de comandos, escriba lo siguiente y presione Intro.

git clone https://github.com/GoogleCloudPlatform/mlops-on-gcp Nota

Si el repositorio clonado no aparece en la IU de JupyterLab, puede usar el menú de la línea superior, y en Git --> Clone a repository, clone el repositorio (https://github.com/GoogleCloudPlatform/mlops-on-gcp) usando la IU.

  1. Verifique que se haya clonado el repositorio, haga doble clic en el directorio mlops-on-gcp y confirme que puede ver su contenido. Los archivos de los labs de Jupyter basados en notebooks de este curso se encuentran en este directorio.
