Aby skopiować notatkę mlops-on-gcp w instancji JupyterLab, wykonaj te czynności:

Krok 1.

W JupyterLab kliknij ikonę Terminal, co spowoduje otwarcie nowego terminala.

Krok 2.

W wierszu poleceń wpisz następujące polecenie i naciśnij Enter.

git clone https://github.com/GoogleCloudPlatform/mlops-on-gcp Uwaga

Jesli skopiowane repozytorium nie pojawia się w interfejsie JupyterLab, możesz użyć menu na górze i skopiować repozytorium za pomocą interfejsu, wybierając Git --> Clone a repository (Skopiuj repozytorium) (https://github.com/GoogleCloudPlatform/mlops-on-gcp).

Krok 3.

Aby sprawdzić, czy repozytorium zostało skopiowane, kliknij dwukrotnie katalog mlops-on-gcp (powinna być widoczna jego zawartość). W tym katalogu znajdują się pliki wszystkich opartych na notatkach modułów Jupyter będących częścią tego szkolenia.
