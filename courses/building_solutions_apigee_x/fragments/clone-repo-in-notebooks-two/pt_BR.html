Siga estas instruções para clonar o notebook mlops-on-gcp na sua instância do JupyterLab:

Etapa 1

No JupyterLab, clique no ícone Terminal para abrir um novo terminal.

Etapa 2

No prompt de linha de comando, digite o código abaixo e pressione Enter.

git clone https://github.com/GoogleCloudPlatform/mlops-on-gcp Observação

Se o repositório clonado não aparecer na IU do JupyterLab, você pode clicar em Git no menu da parte de cima da tela --> Clonar um repositório para realizar o processo (https://github.com/GoogleCloudPlatform/mlops-on-gcp) usando a IU.

Etapa 3

Para confirmar se você clonou o repositório, clique duas vezes no diretório mlops-on-gcp e veja se algum conteúdo aparece. Os arquivos de todos os laboratórios com notebooks do Jupyter deste curso estão nesse diretório.
