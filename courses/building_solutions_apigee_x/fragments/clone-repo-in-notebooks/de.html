So klonen Sie das Notebook training-data-analyst in Ihrer JupyterLab-Instanz:

Schritt 1

Klicken Sie in JupyterLab auf das Terminal-Symbol, um ein neues Terminal zu öffnen.

Schritt 2

Geben Sie folgenden Befehl in die Eingabeaufforderung ein und drücken Sie anschließend die Eingabetaste.

git clone https://github.com/GoogleCloudPlatform/training-data-analyst

Schritt 3

Vergewissern Sie sich, dass das Repository geklont wurde: Wenn Sie auf das Verzeichnis training-data-analyst doppelklicken, sollten Sie den Inhalt sehen. Dieses Verzeichnis enthält alle Dateien für alle Jupyter-Notebook-basierten Labs in diesem Kurs.
