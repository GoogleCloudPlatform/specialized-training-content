Per clonare il blocco note training-data-analyst nella tua istanza di JupyterLab:

  1. In JupyterLab, per aprire un nuovo terminale, fai clic sull'icona Terminal.

  2. Al prompt della riga di comando, esegui il comando seguente:

    git clone https://github.com/GoogleCloudPlatform/training-data-analyst

  3. Per confermare di aver clonato il repository, fai doppio clic sulla directory training-data-analyst e assicurati di poterne vedere il contenuto.
    I file per tutti i lab basati su blocchi note Jupyter di questo corso sono disponibili in questa directory.
