# Vibe Coding for Instant Deployable Apps

## What students learn
Students learn an end-to-end vibe-coding workflow, from UI sketch to functional application. They generate a front-end from a hand-drawn wireframe, add CSV data ingestion and dynamic KPI calculations, integrate live Gemini API calls directly into a running web app, build a structured multi-file React application in Google AI Studio, and apply all of those techniques independently to a use case of their own choosing.

## Case study
Labs 1 through 4 use Cymbal Logistics, a fictional global shipping and supply chain company modernizing its back-office operations. Learners build a Fleet & Shipment Dashboard, a Shift Handoff Tool with AI-generated summaries, an executive slide deck, and a tablet-friendly Driver Log application. In the final lab, students step outside the scenario entirely — defining and building an application of their own design using the full vibe-coding workflow.
