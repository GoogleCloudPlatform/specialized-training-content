# Gemini Enterprise Tools Workshop

## What students learn
Students learn how to use core Gemini Enterprise capabilities in practical workflows. They cover image generation for quick starts, Google Search as a grounding tool, the Canvas tool for visual prototyping, structured prompt engineering, file-based context management, Deep Research, and meta-prompting to improve research quality and repeatability.

## Case study
This course uses Cymbal Capital Partners, a fictional investment firm evaluating potential portfolio companies. Learners use Gemini Enterprise tools to investigate markets, assess founders and company background, and produce stronger investment research inputs. In the final lab, students step outside the scenario entirely — choosing a real challenge from their own work and applying the course tools and techniques independently.

## Labs

| # | Lab | Description |
|---|-----|-------------|
| 1 | [Ice Breaker](1-ice-breaker/tools-ice-breaker.md) | Use Gemini's image generation model to place yourself into a Cymbal Capital Partners investment setting using your own headshot. |
| 2 | [Using the Google Search Tool](2-using-the-google-search-tool/using-google-search.md) | Compare an unsourced Gemini response with a Google Search-grounded one, then run progressively deeper research queries on a real investment theme. |
| 3 | [Vibe-Coding with Gemini Canvas](3-vibe-coding-with-gemini-canvas/vibe-coding-with-gemini-canvas.md) | Translate a hand-drawn UI sketch into a working investment dashboard using Gemini Canvas, then refine it with focused follow-up prompts. |
| 4 | [Prompt Engineering for Investment Analysis](4-prompt-engineering-best-practices/prompt-engineering.md) | Start with a plain request, improve it with the 5-step framework, apply advanced techniques including delimiters and one-shot examples, and use meta-prompting to improve your own prompts. |
| 5 | [Managing Files as Context](5-managing-files-as-context/managing-files-as-context.md) | Add PDFs, spreadsheets, DOCX files, and images to Gemini Enterprise and see how the quality of the investment analysis changes as the file set grows and shrinks. |
| 6 | [Deep Research for Investment Due Diligence](6-deep-research/deep-research.md) | Use Gemini Enterprise Deep Research to investigate a company across three passes — overview, founders, and market viability — then synthesize the findings into a partner-ready investment note. |
| 7 | [Meta-Prompting for Deep Research](7-meta-prompting-for-deep-research/meta-prompting.md) | Run a vague prompt through Deep Research, use meta-prompting to let Gemini improve it, then run the stronger version and compare the results. |
| 8 | [Bring Your Own Use Case](8-bring-your-own-use-case/bring-your-own-use-case.md) | Apply the full Gemini Enterprise toolkit independently — choose the right tool for a real challenge from your own work, build the output, and refine it based on a critical review. |
